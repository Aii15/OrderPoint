# OrderPoint2

Monorepo scaffold for a cafe kiosk platform.

## Structure

- `apps/kiosk`: Next.js customer kiosk app
- `apps/admin`: Next.js dashboard for owner/staff
- `apps/kds`: lightweight Next.js kitchen display
- `apps/api`: NestJS backend
- `packages/types`: shared TypeScript interfaces
- `packages/ui`: shared UI package (optional)
- `packages/config`: shared config package
- `docker`: container and nginx configs

This commit provides the requested folder and file scaffold.
